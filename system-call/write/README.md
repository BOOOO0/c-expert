# write

- write 함수는 unistd.h 헤더 파일에 정의된 쓰기 함수입니다.

- 함수의 원형은 다음과 같습니다.
  ```C
  ssize_t write(int fd, const void* buf, size_t n);
  ```
  버퍼에 n만큼의 바이트를 쓰고 

## 파일 디스크립터(fd)

- write함수에서 파일 디스크립터는 write할 대상을 의미합니다. 정해진 번호를 사용할 수도 있고 대상이 될 파일을 명시할 수도 있습니다.

- |fd|의미|
  |:-:|:-:|
  |0|표준 입력|
  |1|표준 출력|
  |2|에러 출력|
