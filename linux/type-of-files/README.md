# 파일의 종류

- 리눅스에서 제공하는 파일의 종류는 다음과 같습니다.

## 일반 파일

- 텍스트 파일, 멀티미디어 파일 등이 일반 파일로 분류됩니다.

## 디렉토리

- 디렉토리도 파일의 한 종류이며 윈도우에서의 폴더와 같은 개념입니다.

## 심볼릭 링크 파일

- 다른 파일을 가리키는 파일이며 윈도우의 바로가기 아이콘과 같은 개념입니다.

## 디바이스 파일

- 디바이스 파일은 유저 모드에서 응용 프로그램이 동작할 때 커널 모드에서 OS가 디바이스 드라이버를 통해 하드웨어를 제어하는 것 처럼 하드웨어를 제어하기 위해 운영체제 수준에 있는 디바이스 드라이버와 응용 프로그램 사이의 통로 역할을 합니다.

- 디바이스 파일은 동작 방식에 따라서 블록 디바이스 파일과 문자 디바이스 파일로 나뉩니다.

## 문자 디바이스 파일

- 키보드의 입력 처럼 순차적으로 입력되는 장치에 대한 제어를 문자 디바이스 파일이 담당합니다.

## 블록 디바이스 파일

- 메모리의 영역처럼 영역에 대한 공유를 제어할 때 블록 디바이스 파일을 사용합니다.

## 파이프 파일

- 프로세스와 프로세스의 통신(IPC)에 사용되는 파일로 프로세스가 파이프를 통해 통신을 한다면 파이프 파일이 존재한다고 생각할 수 있습니다.

## 소켓

- 프로세스간 통신에 사용되며 소켓의 종류에 따라 실제 파일로 존재할 수도 있습니다.
